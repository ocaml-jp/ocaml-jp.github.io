9.6 オプション

ocamlmktop は次のようなコマンドラインオプションを認識します。

-cclib libname

カスタムランタイムモードでリンクするときに C リンカにオプションとして -llibname を渡します。 8 章「バッチコンパイル（ocamlc）」 の ocamlc の同名のオプションを参照してください。

-ccopt option

カスタムランタイムモードでリンクするときに C コンパイラやリンカに指定されたオプションを渡します。 8 章「バッチコンパイル（ocamlc）」 の ocamlc の同名のオプションを参照してください。

-custom

カスタムランタイムモードでリンクします。 8 章「バッチコンパイル（ocamlc）」 の ocamlc の同名のオプションを参照してください。

-I directory

指定されたディレクトリをコンパイル済みオブジェクトコードファイル（.cmo ファイルや .cma ファイル）を探索するディレクトリのリストに追加します。

-o exec-file

リンカの出力するトップレベルファイルの名前を指定します。 デフォルトは a.out です。
