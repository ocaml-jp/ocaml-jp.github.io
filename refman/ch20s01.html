20.1 Conventions

リファレンス用に標準モジュール群をアルファベット順で以下に挙げます。 各モジュールに対して、シグネチャを使った宣言をタイプライタ体でひとつずつ示し、それぞれに簡単な説明をつけます。 モジュールとそれらが公開している識別子は本報告書の末尾の索引を見てください。

  • Arg モジュール: コマンドライン引数の解析
  • Array モジュール: 配列の操作
  • ArrayLabels モジュール: 配列の操作（ラベル付き）
  • Buffer モジュール: 伸張可能な文字列バッファ
  • Callback モジュール: Caml の値の C ランタイムへの登録
  • Char モジュール: 文字の操作
  • Complex モジュール: 複素数
  • Digest モジュール: MD5 メッセージタイジェスト
  • Filename モジュール: ファイル名の操作
  • Format モジュール: pretty printing
  • Gc モジュール: memory management control and statistics; finalised values
  • Genlex モジュール: 汎用字句解析器
  • Hashtbl モジュール: ハッシュテーブルとハッシュ関数
  • Int32 モジュール: 32 ビット整数
  • Int64 モジュール: 64 ビット整数
  • Lazy モジュール: 遅延計算
  • Lexing モジュール: the run-time library for lexers generated by ocamllex
  • List モジュール: リストの操作
  • ListLabels モジュール: ファイル名の操作（ラベル付き）
  • Map モジュール: 順序型上の対応表
  • Marshal モジュール: データ構造のマーシャリング
  • MoreLabels モジュール: Include modules Hashtbl, Map and Set with labels
  • Nativeint モジュール: プロセッサネイティブな整数
  • Oo モジュール: オブジェクト指向拡張
  • Parsing モジュール: the run-time library for parsers generated by ocamlyacc
  • Printexc モジュール: 例外の表示
  • Printf モジュール: 書式付き表示関数
  • Queue モジュール: FIFO キュー
  • Random モジュール: 疑似乱数生成器
  • Scanf モジュール: formatted input functions
  • Set モジュール: 順序型上の集合
  • Sort モジュール: sorting and merging lists
  • Stack モジュール: LIFO スタック
  • StdLabels モジュール: ラベル付きの Array、 List、 String
  • Stream モジュール: ストリームと解析器
  • String モジュール: 文字列の操作
  • StringLabels モジュール: 文字列の操作（ラベル付き）
  • Sys モジュール: システムインタフェース
  • Weak モジュール: arrays of weak pointers
